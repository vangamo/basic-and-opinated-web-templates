Este repo incluye una colección de templates de proyectos y carpetas con código listo para usar, muy sencillo pero bien estructurado.

## Express server

Un servidor web usando [Express](https://github.com/expressjs/express) que implementa:

- Un API RESTful
- Un servidor de ficheros estáticos para los ficheros build de React
- Un servidor de ficheros dinámicos usando el motor de plantillas [EJS](https://github.com/mde/ejs)
- Autenticación y autorización con tokens JWT y contraseñas almacenadas en una base de datos
- Arquitectura modelo-vista-controlador
- Test unitarios usando [Jest](https://github.com/jestjs/jest) y [Supertest](https://github.com/forwardemail/supertest)
- Test de integración usando [Playwright](https://github.com/microsoft/playwright)
- Documentación del API usando [Swagger](https://github.com/swagger-api/swagger-ui) (pendiente...)
