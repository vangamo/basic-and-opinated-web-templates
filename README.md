A collection of project templates and boilerplates very basic but well-structured.

## Express server

A web server using [Express](https://github.com/expressjs/express) implementing:

- A RESTful API
- A server for static React files
- A server for dynamic files using [EJS](https://github.com/mde/ejs)
- Authentication and authorization with JWT tokens and password stored in a database
- MVC architecture
- Unit tests using [Jest](https://github.com/jestjs/jest) and [Supertest](https://github.com/forwardemail/supertest)
- Integration tests using [Playwright](https://github.com/microsoft/playwright)
- Documentation using [Swagger](https://github.com/swagger-api/swagger-ui) (pending...)
